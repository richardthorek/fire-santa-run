## Interaction States

### NEXT Button
- Default: Orange background, black text
- Tap/Press: Scale down to 95% (transform: scale(0.95))
- Release: Scale back to 100%

### Arrived Button
- Disabled: Gray, cursor: not-allowed
- Enabled: Green gradient, cursor: pointer
- Tap: Completes waypoint, advances to next

### Stop Button  
- Always enabled
- Tap: Shows confirmation, exits navigation

## Key User Benefits

1. **Always know what's next** - See upcoming waypoint before arriving
2. **Manual control** - Skip waypoints when needed (e.g., no one home)
3. **Driver-friendly** - Large buttons, clear text, easy to tap while stopped
4. **Flexible completion** - Auto-complete (proximity) OR manual (Arrived button) OR skip (NEXT button)
5. **Clear status** - Stop counter shows progress (2/5 stops)
